### 子比主题自动翻译插件
- 插件名字为 Quicksleep
- 插件用途是可以在让玩家数量到一定程度就睡觉
  
### 插件介绍
- 多语言
- 可以自定义国旗样式
- 设置翻译按钮样式
- 简单的插件
- 自动更新

### 使用方法
- 到Release下载解压
- 把文件夹放到plugins目录
- 后台启用
- 在后台设置
